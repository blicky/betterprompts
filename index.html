Tools for creating better prompts for AI

Disclosure: I did not try all of them in detail, consider this more like a public bookmarking list

  • MidJourney Prompt Helper
    Create MJ prompts with a simple editor using different options.
    https://prompt.noonshot.com/
  • Prompting Guide
    Explanations and guides on how to better engineer prompts
    https://www.promptingguide.ai/
  • Prompts.ai
    Improve prompts with different parameters like temperature and selection of the model
    https://prompts.ai/
  • PromptPerfect
    Prompt optimizer tool for ChatGPT, Dall-E, Midjourney and Lexica
    https://promptperfect.jina.ai/
  • Promptify
    Improve prompts for GPT based on your input
    https://promptify.pro/
  • Chatbot UI
    Open Source improved UI for GPT. Still in early days. Watch twitter post for more details.
    https://github.com/mckaywrigley/chatbot-ui
  • ChatGPT3 Prompt Engineering
    Guide and framework for creating ChatGPT3 Prompts.
    https://github.com/mattnigh/ChatGPT3-Free-Prompt-List
  • Talk to Transformer
    A web app that lets you generate text using GPT-2, OpenAI's previous language model.
    https://talktotransformer.com/
